• ngx-page-view

    What is this?

    You can implement tiktok like swipe view easily with ngx-page-view

    • available vertical and horizontal swipe
    • availabel touch swiping / mouse swipiping / mouse scrolling
    • Customizable

    Example

    How to customize?

    You can customize following property

    • threshold (default 0.3)
    • direction (default PageViewDirection.horizontal): swipe direction
    • current (default: 0): if you pass current index of page view, then it will jump to the page
